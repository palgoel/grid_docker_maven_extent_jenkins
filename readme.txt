In maven build add two params:It can be of any value you like
browser="Firefox"
url="https://www.google.com"
This project is complete selenium java maven framework with Extent report. I am adding selenium grid capability with docker compose pom.xml takes two inputs: browser and url. This is basic POM maven framework with java selenium.It just has basic testcases like check title and current url etc. Main aim of this repository is about understanding of selenium grid with maven framework and generation of extent report. Later I will intergrate this with jenkins also.
